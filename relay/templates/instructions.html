Tally Light Setup Guide

How to Use

Follow these steps carefully to set up your Tally Lights.

  1. Read the entire instructions first.
  2. Review the color explanations and troubleshooting sections.
  3. Determine which mode you want to use.
  4. Modes of operation:
    • A — Direct connection to ATEM
    • B — Connection via Relay Server + .exe program

Mode A — Direct ATEM connection

  1. Power up the ATEM and connect it to the internet via Ethernet.
  2. Power up the Tally Lights.
  3. Flip the switch to enable SOFT AP; the LED must turn white.
  4. On your phone, connect to a Wi-Fi network like “tally setup” or “ESP-…”.
  5. Open a browser and go to 192.168.4.1.
  6. Fill in all required settings, including ATEM IP. Ensure Use Relay Server is set to FALSE.
  7. Make sure Tally Light's Wi-Fi network is the same as ATEM's
  8. Click Save. You’re ready to go.

Mode B — Relay Server

  1. Power up the ATEM and connect it to the internet via Ethernet.
  2. On a computer connected to the same network as ATEM, open this link
  3. Download and run the .exe file.
  4. Ensure antivirus or Windows Defender is not blocking it.
  5. When prompted, fill the required values or click ENTER to use default ones.
    • Relay Room ID is a unique identifier used to pair Atem and Tally Lights together. Make sure to MEMORIZE IT
  6. Confirm the program shows: Connected to Atem Switcher.
  7. Flip the switch to enable SOFT AP; the LED must turn white.
  8. Open a browser and go to 192.168.4.1.
  9. Fill in all required settings.
  10. Ensure Use Relay Server is set to TRUE.
  11. Ensure Room ID is the same as the one used in step 5.
  12. Click Save. You’re ready to go.

Explanation of Colors

Blue
Connecting to Wi-Fi.
White
SOFT AP mode active.
Pink
Connecting to ATEM or Relay.
Yellow
Connected but no tally data.
Green
PREVIEW.
Red
LIVE.
Off
Inactive or idle.

Troubleshooting

White LED is ON, but no setup Wi-Fi is visible

Possible causes:

  • Your device is caching Wi-Fi scan results
  • Your device refuses to recognize the setup Wi-Fi as a proper Wi-Fi network

Solution:

  • Turn Wi-Fi off on your device
  • Flip the switch to SOFT AP mode
  • Wait up to 30 seconds and scan for Wi-Fi networks

Blue LED is ON, and Wi-Fi credentials are 100% correct, but the Tally Light never connects

Possible causes:

  • Wrong Wi-Fi credentials
  • The Wi-Fi network must be 2.4 GHz, NOT 5 GHz for the Tally Light to connect

Solution:

  • Verify the ATEM IP address
  • Ensure the .exe program shows “Connected to Atem Switcher”
  • Confirm the Relay Room ID matches exactly (case-sensitive)

My device sees the setup network, tries to connect to it, but the connection fails or does not last long

Possible causes:

  • Poor network stability
  • Your device sees “better” networks and automatically connects to them instead
  • Your device refuses to stay connected because the network has no internet access

Solution:

  • Disable auto-reconnect in your device’s Wi-Fi settings
  • Manually force the connection to the setup network

LED stays yellow for a long time

Meaning: Tally is connected to Wi-Fi and the relay server, but receives no tally data.

Solution:

  • Check whether the .exe program is running
  • Verify the relay server settings
  • Restart both the Tally Light and the .exe program

Charging

Charge via USB-C for 2–4 hours. Red internal LED indicates charging. Estimated battery life: ~20 hours.
